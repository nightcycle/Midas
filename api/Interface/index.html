Interface

The front-facing module used to integrate a game with the Midas library. Returns a midas configured for that path. If one already exists for that path it will provide that one.

This item only works when running on the server. Server
</>

Provides a pseudo-RBXScriptSignal which will fire whenever an event is sent via HttpService. When the Signal is fired it will provide the playerId, path, data, tags, and timestamp in that order.
